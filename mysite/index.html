Project Site Plan for BizlyHub

  1. Site Name

    BizlyHub
    This name represents the business and the service it provides. Biz short for business and ly for modern feel. Hub represents a central location for web development services.
    Domain: BizlyHub.com

  2. Site Purpose

    for the BizlyHub Webiste
    The site provides comprehensive website services, from design and development to hosting and maintenance, helping businesses establish and grow their online presence.

  3. Scenarios

    for the BizlyHub Webiste


    • I'm a small business owner and I need a website, but I don't know where to start. I need something affordable, but it has to look professional and work well on mobile phones. Can BizlyHub help me with that?
    • I already have a website, but it's old and doesn't get much traffic. Does BizlyHub offer services to improve my website's search engine ranking?

  4. Color Schema

    Primary Color: #9CE5FF (Light Blue)
    Usage:

    • Background color for main sections of the website.
    • Accent color for buttons, links, and interactive elements.
    • Background color for headers or footers (depending on design).

    Secondary Color: #000000 (Black)
    Usage:
    • Text color for all headings, paragraphs, links, etc. Essentially, all text content.
    • Borders or dividers.
    • Buttons

    Accent Color: #FFFFFF (White)
    Usage:
    • Page background.
    • Text color on black buttons.
    • Cards and containers

  5. Typography

    Primary Font: Roboto (Sans-Serif)
    Usage:

    • Headings (H1, H2, H3, H4, H5, H6, P)
    • Buttons & Navigation
    • Call-to-action (CTA) elements

    Secondary Font: Serif (Backup for Roboto)
    Usage:
    • Body text (paragraphs, long-form content)
    • Any fallback when Roboto isn't available
